# Mid-term-assignment-GPT2
This is the mid-term assignment of my AI course. The training set is the poetry set of Naran Rongru. 

Run test.py and you can get 10 poems：

```bash
python test.py
```
An example of the result is shown below：

```bash
输入提示： 江梅
```
